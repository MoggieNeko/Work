# Extension 使用說明

## 安裝

1. 打開 `chrome://extensions/`
2. 開啟「開發人員模式」
3. `載入未封裝項目`
4. 選擇本資料夾 `extension/`

## 使用

### Hashtag 模式

- query 輸入 `coffee` 或 `latteart`
- extension 會打開對應 hashtag 頁並捲動收集

### Keyword 模式

- query 輸入一般字詞，例如 `coffee review`
- extension 會嘗試在 Instagram 搜尋框輸入並點入結果
- 若失敗，建議改用 hashtag 模式，或先手動打開結果頁再用 collect 模式

### Collect 模式

- 當你已經在 Instagram 某個結果頁
- extension 只做捲動與收集，不嘗試搜尋
